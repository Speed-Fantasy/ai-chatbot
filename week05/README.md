# 📝 AI 简历优化器

上传简历，AI 从四个维度打分分析，支持 JD 匹配、STAR 改写、报告导出。

## 功能

- 📤 上传简历 — 支持 PDF / DOCX / TXT
- 📊 四维 AI 分析 — 完整性、措辞力度、关键词、结构清晰度
- 🎯 JD 匹配 — 对比岗位要求，找出差距
- ✍️ AI 改写 — STAR 法则优化，量化成果
- 📝 一键导出 — Markdown 格式优化报告
- 💾 数据持久化 — 关闭浏览器不丢失

## 技术栈

```
Streamlit + DeepSeek API + PyPDF2 + python-docx
```

## 运行

```bash
pip install -r requirements.txt
streamlit run app.py
```
