# 两个月 AI 应用开发速成计划

**时间：** 2026年7月 - 8月  
**目标：** 产出 3-4 个可上线的 AI 应用项目 → 写进简历 → 找实习

---

## 总体策略

- 每天 4-6 小时，每周 6 天，周日复盘整理
- 不学纯理论，需要用到的知识当场学
- 每个项目推到 GitHub 并部署上线
- 全程用 Claude/Cursor 辅助写代码

---

## 技术栈

`Python` `Streamlit` `Claude API` `ChromaDB` `LangChain` `CrewAI`

---

## 学习路径（由浅入深）

| 层级 | 做什么 | 难度 |
|------|--------|------|
| 调用 API 层 | 调 Claude/GPT API，给 prompt，拿结果 | ★ |
| 工作流编排层 | 多个 API 调用串起来，加判断逻辑 | ★★ |
| Agent 层 | AI 自主决策、使用工具、多步执行 | ★★★ |
| RAG 层 | 向量数据库 + Embedding，知识库问答 | ★★★ |
| 微调/训练层 | 暂不涉及 | ★★★★★ |

## 第五周：完整 AI 产品——三个方向选一个

| 选项 | 选题 | 说明 |

## 避坑指南

| ❌ 别做 | ✅ 要做 |
|---------|--------|
| 别从头啃机器学习数学 | 数学很重要但不是入门需要的 |
| 别一上来就学 PyTorch 训练模型 | 你是应用开发，不是算法研究 |
| 别看完教程才动手 | 看完第一章就开始写 |
| 别纠结选什么框架 | 就用 LangChain，别人说什么不重要 |
| 别只追求完美 | 每个项目必须推到 GitHub 并部署上线 |
